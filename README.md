\# StarTask — зарабатывай Telegram Stars



StarTask — платформа микрозаданий внутри Telegram, где рекламодатели платят Stars за реальные действия пользователей.



\## Технологии



\- Backend: Node.js + Express + PostgreSQL

\- Frontend: React + Vite (Telegram Mini App)

\- Bot: Telegraf.js



\## Переменные окружения



\- `DATABASE\_URL` — PostgreSQL (Neon.tech)

\- `BOT\_TOKEN` — токен от @BotFather

\- `JWT\_SECRET` — секрет для JWT

\- `MINI\_APP\_URL` — URL деплоя Mini App



\## Грант TON Foundation



Проект подан на грант TON Foundation для масштабирования.
